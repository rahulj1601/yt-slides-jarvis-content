The Process

Four steps, fully automated

Step 01  —  Saturday 05:30

Idea Generation

Cron triggers the YouTube + LinkedIn + Newsletter agents. They analyse recent performance data and generate a week of content ideas - ranked by topic authority and relevance.

Auto-run
Step 02  —  On demand

Scripting

How to build yours

Start small. One workflow.
One skill. Then stack.

Build order

✓

Week 1: Set up one agent with your brand voice and one skill (e.g. LinkedIn post writer).

✓

Week 2: Wire in a live data source (Trello, Notion, or a Google Sheet).

✓

Week 3: Add a cron job. Let it run once a week unsupervised.

✓

Week 4+: Add agents. Specialise. Connect them. Build the stack.

Principle 01

One agent, one job

Specialist agents outperform generalist ones. LinkedIn agent only thinks about LinkedIn.

Principle 02

Feed it real data

Agents that read your actual analytics write 10x better content than ones working from a prompt alone.

Principle 03

Automate the schedule

A cron job beats willpower. Set a Saturday trigger and the system runs whether you wake up or not.
